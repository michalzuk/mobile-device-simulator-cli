# Security Policy

## Supported Versions

Only the latest published version is supported with security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities by email to: `security@michalzuk.dev`

Include the following details:

- A clear description of the issue
- Reproduction steps or proof of concept
- Expected vs actual behavior
- Potential impact

## Response Targets

- Initial acknowledgement: within 72 hours
- Triage decision: within 7 days
- Status updates: at least weekly until resolved

If the report is valid, a fix will be prioritized and released as soon as possible.
